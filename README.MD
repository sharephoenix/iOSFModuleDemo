# 工程模块化

## 工程目录介绍

1. Demo -> 一个项目工程
2. Vendor -> 一个项目工程
3. YSDymicLibrary YSStaticLibrary 是两个静态库，可以理解为以后写的组件
4. MyWorkspace.xcworkspace 单独创建的工程管理目录
5. Podfile 使用 cocoapods 管理项目的公共依赖
6. Cartfile 使用 Cartage 管理项目依赖（各种原因体验失败，可能我太菜）
